## Installation
1. Clone repository
2. Install [npm i node](https://www.npmjs.com/package/node), [npm i inquirer](https://www.npmjs.com/package/inquirer), and if testing [npm i jest](https://www.npmjs.com/package/jest)
3. Run 'npm start' in your terminal
