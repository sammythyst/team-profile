## Installation
Clone repository
Install [npm i node](https://www.npmjs.com/package/node), [npm i inquirer](https://www.npmjs.com/package/inquirer), and if testing [npm i jest](https://www.npmjs.com/package/jest)
Run 'npm start' in your terminal
